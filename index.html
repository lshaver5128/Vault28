Our Mission

“Vault 28 Trading Co. was created to make selling sports cards easier, honest, and highly professional. 🛠️ Connect Google Firebase

Step 1: Go to Firebase Console Open console.firebase.google.com and sign in with your Google account.
Step 2: Create a Project Click Add Project, name it (e.g. Vault28), disable Google Analytics (recommended for fast setup), and click Create.
Step 3: Register a Web App In your project overview, click the Web icon (</>) to register an app. Name it Vault28Web and click register.
Step 4: Copy Config Keys Firebase will show you a configuration script. Copy only the firebaseConfig object. It looks like this: 
const firebaseConfig = {
  apiKey: "AIzaSy...",
  authDomain: "vault28.firebaseapp.com",
  projectId: "vault28",
  storageBucket: "vault28.appspot.com",
  messagingSenderId: "123456...",
  appId: "1:123456..."
};
Step 5: Paste in app.js Open app.js in VS Code and paste your config into the FIREBASE_CONFIG variable at the very top of the file. Save and push the change to your GitHub repo!
